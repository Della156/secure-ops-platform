# Trae AI 设计提示词：模块1 — 网络安全自动任务配置

## 项目概况

现有 Next.js 项目，基于 TailwindCSS + TypeScript，已具备：
- 6个一级目录菜单（侧边栏）
- 三层菜单数据结构支持（MenuItem → SubMenuItem → ThirdMenuItem 类型已定义）
- 当前在 `menuData.ts` 中已为「网络安全自动任务配置」配置了 14 个二级目录和 57 个三级菜单

## 你的任务

为 **第一个一级模块「网络安全自动任务配置」** 下的 **每一个三级菜单页面** 设计实现前端页面。

### 2. 组件风格
- 卡片容器：`bg-slate-900 border border-slate-800 rounded-xl`
- 输入框：`bg-slate-800 border border-slate-700 rounded-lg text-white`
- 按钮（主）：`bg-blue-600 hover:bg-blue-700 text-white rounded-lg px-4 py-2`
- 按钮（次）：`bg-slate-800 hover:bg-slate-700 text-slate-300 rounded-lg px-4 py-2`
- 表格：`w-full border-collapse` 表头 `bg-slate-800/50 text-slate-400` 行 `hover:bg-slate-800/30`
- 标签/徽章：`bg-blue-500/20 text-blue-400`、`bg-green-500/20 text-green-400` 等
- 搜索框：左侧搜索图标 + 输入框，`bg-slate-800 border border-slate-700`
- 分页：底部居中，页码按钮 `bg-slate-800 hover:bg-slate-700`

### 3. 页面通用布局
每页布局结构（从上到下）:
```
┌─ 页面标题（二级/三级菜单名称）
├─ 操作栏（搜索框 + 新增按钮 + 批量操作等）
├─ 筛选区（下拉选择器、日期范围等）
├─ 数据表格（列表展示）
├─ 分页（底部）
└─ 弹窗（新增/编辑/详情 Modal，使用 Dialog）
```

### 4. 状态管理
- 使用 `useSystem().activeMenu` 获取当前激活菜单
- 菜单路由 ID 格式：`menu-1-x-y`（一级-二级-三级）
- 当前页面用 `{children}` 渲染在 `MainLayout` 中

### 5. 通用功能模式
所有 CRUD 页面通用的操作模式：
- **新增**：右上角「+ 新增」按钮 → 弹出 Modal 表单 → 填写 → 提交 → 刷新列表
- **编辑**：表格行操作列「编辑」按钮 → 弹出 Modal 表单（带初始值）→ 修改 → 提交
- **删除**：表格行操作列「删除」按钮 → Confirm 确认 → 删除 → 刷新列表
- **查询**：顶部搜索框 + 筛选条件 → 点击「查询」→ 更新列表
- **导出**：列表顶部「导出」按钮 → 导出为 Excel/CSV
- **详情查看**：点击行或「详情」按钮 → 弹出详情 Modal 或跳转详情页

## 功能实现要求

请为每个三级菜单创建独立的页面组件，放在 `src/pages/module1/` 目录下（或根据项目路由约定）。每个页面要实现：

### 第1组：自动化任务注册与接入（3页）

#### 1.1 任务接入管理
- **功能列表**：任务接入配置的表格展示（ID、任务名称、接入协议类型、接入参数、接入状态、创建时间、操作）
- **新增/编辑**：Modal 表单，字段包括：任务名称、接入协议（下拉选择：SSH/REST/SNMP/SQL）、接入参数（JSON编辑器或键值对）、描述
- **查询**：按任务名称/接入协议/状态筛选
- **操作**：编辑、删除、连接测试

#### 1.2 任务在线注册管理
- **功能列表**：注册任务列表（ID、任务名称、注册编号、注册状态[已注册/已注销]、审核状态、申请时间）
- **注册流程**：注册申请表单（任务信息、接入信息、申请说明）→ 提交后状态流转
- **审批流程**：审批配置的查看（审批节点、审批人、审批结果）
- **操作**：注册申请、查看审批流程、注销

#### 1.3 任务接入状态管理
- **状态展示**：卡片式/列表式展示各任务的接入状态（正常/异常/离线）
- **监控面板**：状态概览（总数、正常数、异常数）、实时状态指示器
- **历史查询**：接入状态变更历史时间线/表格
- **告警提示**：异常状态自动高亮、告警标记
- **操作**：手动刷新状态、查看变更记录

### 第2组：自动化任务仓库与管理（2页）

#### 2.1 任务版本管理
- **列表**：表格展示（任务名、版本号、版本描述、状态、创建时间、操作）
- **版本操作**：新增版本、版本回滚、版本审核流程展示、版本发布/下线
- **差异对比**：选中两个版本进行 diff 对比显示（左右对比视图）

#### 2.2 任务上下架管理
- **列表**：表格展示（任务名、当前状态[已上架/已下架]、上架时间、下架时间、操作人）
- **上架流程**：上架申请 → 上架审批流程展示
- **下架流程**：下架申请 → 下架审批流程展示
- **操作**：申请上架、申请下架、查看日志

### 第3组：自动化任务运行状态监控（6页）

#### 3.1 任务运行状态列表
- **列表**：表格（任务ID、任务名称、运行状态[运行中/成功/失败/等待]、开始时间、结束时间、耗时、操作）
- **筛选**：按状态、时间范围、任务名称筛选
- **导出**：导出当前列表为 Excel

#### 3.2 任务运行统计分析
- **统计维度**：日/周/月切换
- **图表**：执行次数柱状图、成功率趋势折线图、平均耗时分布图
- **报表**：统计报表导出按钮

#### 3.3 任务运行监控
- **实时监控**：任务执行进度条展示、关键节点耗时甘特图
- **日志**：执行日志实时滚动查看面板（暗色终端风格）
- **布局**：左侧进度总览，右侧日志详情

#### 3.4 任务资源使用率监控
- **指标**：CPU、内存、网络 IO 的实时仪表盘或折线图
- **阈值告警**：阈值配置表单（新增/编辑/删除告警规则）
- **趋势**：历史趋势图（可选择时间范围）

#### 3.5 任务异常事件分析
- **列表**：异常事件列表（异常ID、任务名、异常类型、异常原因、发生时间、状态）
- **分析**：异常原因智能分析展示卡片
- **统计**：按异常类型分类的饼图/柱状图
- **关联**：点击查看关联的执行日志

#### 3.6 任务告警管理与推送
- **规则管理**：告警规则表格（规则名称、阈值条件、告警级别、状态、操作）
- **级别定义**：告警级别标签管理（新增/编辑级别名称、颜色、优先级）
- **接收人**：告警接收人配置（多选用户）
- **方式**：告警方式配置（邮箱/短信/ELINK等）
- **历史**：告警历史记录查询表格

### 第4组：可编排任务目录（3页）

#### 4.1 自动化能力可视化检索与浏览
- **目录展示**：网格/列表展示任务卡片（名称、标签、分类、简要描述）
- **检索**：搜索框 + 标签/分类筛选器（模糊搜索）
- **详情**：点击卡片展开详情（功能描述、输入/输出参数表格、调用示例代码块）
- **调用示例**：代码块展示（支持复制）

#### 4.2 任务服务接口授权配置
- **策略列表**：表格（策略名称、授权对象[角色/用户]、状态、有效期、操作）
- **授权流程**：新增授权策略 → 选择角色/用户 → 设置有效期
- **申请/审批**：授权申请列表 + 审批操作按钮

#### 4.3 任务接口描述与文档查看
- **文档列表**：接口列表（接口名称、版本、更新时间、操作）
- **在线预览**：Swagger/OpenAPI 风格文档预览（可折叠的接口详情）
- **版本管理**：版本切换、版本对比
- **下载**：文档下载（JSON/YAML）

### 第5组：自动化流程编排器（4页）

- **节点配置**：点击节点弹出配置侧边面板
- **导入/导出**：流程 JSON 导入导出

#### 5.2 自动化能力节点库
- **分类展示**：左侧分类树/标签，右侧节点网格
- **搜索**：按名称搜索节点
- **详情**：点击节点查看详情（功能描述、参数说明）
- **拖拽**：节点可拖拽至编排画布

#### 5.3 流程逻辑控制节点

- **连通性监测**：状态指示器（在线/离线），最近监测时间

#### X.3 [资源]访问授权管理
- **授权列表**：表格（授权策略名、授权对象[角色/用户]、状态、有效期、操作）
- **授权配置**：新增/编辑授权策略（选择角色/用户、设置有效期）
- **申请/审批**：授权申请列表 + 审批操作（通过/拒绝）

#### X.4 [资源]可用服务/指令/API视图
- **服务/指令/API列表**：表格（名称、类型、描述、参数说明）
- **示例**：调用示例代码块（可复制）
- **详情**：展开查看详细参数

#### X.5 [资源]访问日志查询
- **日志列表**：表格（时间、用户、操作类型、目标资源、结果、详情）
- **筛选**：按资源/用户/操作类型/时间范围查询
- **详情**：点击查看日志详情
- **导出**：导出日志

### 第13组：数据接口对接管理（4页）

#### 13.1 数据接口配置管理
- **列表**：表格（接口名称、接口类型、连接状态、创建时间、操作）
- **新增/编辑**：Modal（接口名称、接口URL、认证方式、连接参数键值对、数据格式映射配置）
- **查询**：按名称/类型筛选

#### 13.2 数据接口连接测试验证
- **测试**：选择接口 → 点击「连接测试」→ 显示测试结果
- **数据预览**：测试成功后预览返回数据
- **诊断**：连接状态诊断信息展示

#### 13.3 数据接口调用日志查询
- **列表**：日志表格（调用时间、接口、调用状态、耗时、结果）
- **筛选**：按接口/时间/状态查询
- **详情**：查看调用详情和失败原因分析

#### 13.4 数据接口调用性能分析
- **指标**：响应时间统计（平均/最大/最小耗时折线图）
- **趋势**：调用次数趋势图
- **导出**：分析报告导出

### 第14组：自动化服务接口配置管理（4页）

#### 14.1 任务API接口配置
- **列表**：API接口列表（API名称、路径、请求方法、状态、操作）
- **配置**：选择自动化任务 → 发布为API → 配置路径、方法（GET/POST/PUT/DELETE）、输入输出参数映射

#### 14.2 任务API访问授权管理
- **AppKey管理**：AppKey/AppSecret 列表（应用名、AppKey、状态、创建时间）
- **授权策略**：IP白名单配置（新增/编辑/删除）

#### 14.3 任务API调用日志查询
- **日志**：表格（调用时间、应用、API、状态码、耗时）
- **筛选**：按应用/API/时间/状态查询
- **详情**：请求/响应详情查看

#### 14.4 任务API接口调用分析与统计
- **指标**：调用次数、成功率、平均耗时统计卡片
- **维度**：按时间/应用维度切换
- **可视化**：折线图/柱状图展示
- **报表**：导出报表
